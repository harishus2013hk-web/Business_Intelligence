# Healthcare Analytics Dashboard

A comprehensive **Healthcare Dashboard** built to analyze hospital operations, patient demographics, treatment performance, departmental metrics, and comparative healthcare KPIs through interactive multi-page reporting.

---

## Project Overview

This dashboard provides end-to-end visibility into hospital and patient data across **4 interactive report pages**, enabling healthcare administrators and analysts to monitor performance, optimize resource allocation, and improve patient outcomes.

---

# Dashboard Pages

## 1. Hospital Summary

Provides a high-level overview of hospital performance and operational metrics.

### Key Metrics

* Total Patients
* In Patients / Out Patients
* Total Staff
* Average Treatment Cost
* Death %
* Readmission %
* Occupied Beds

### Insights Included

* Monthly Inpatient vs Outpatient Trends
* Revenue by Age Group
* Bed Occupancy Gauge
* Revenue by Month
* Department-wise ER Time
* State/City Performance Analysis
* Readmission & Death Analysis by Age Group

---

## 2. Patient Summary

Focused on patient demographics, admissions, and treatment insights.

### Key Metrics

* Total Patients
* Admitted / Discharged
* Average Length of Stay (LOS)
* ICU Patients
* Death Count
* Available / Occupied Beds

### Insights Included

* Gender Distribution
* LOS by Age Group
* Patient Status by Gender
* Age Group Funnel Analysis
* Geographic Patient Distribution Map
* Department-wise Ratings & Positive Feedback

---

## 3. Patient Details (Drill Through)

Detailed transaction-level patient records for deep analysis.

### Features

* Searchable Patient-Level Table
* Multi-Dimensional Filtering
* Drill-through Navigation from Summary Pages

### Data Available

* Patient Name
* Age / Gender
* Bed Number / Bed Status
* State / City
* Department
* Staff Name
* Admission Type
* Patient Status

---

## 4. Period on Period Comparison

Compare healthcare KPIs across current vs previous periods.

### Key Metrics

* Revenue Growth %
* Death % Comparison
* Patient Growth Trends

### Insights Included

* Current vs Previous Revenue Comparison
* Current vs Previous Patient Counts
* Growth Analysis by Dimension (Gender, State, Department, etc.)
* Revenue Growth Visualization
