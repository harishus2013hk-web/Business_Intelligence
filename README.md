# 📊 Power BI Dashboard: [Sales Analysis]

This repository contains a Power BI dashboard designed to visualize and analyze [brief description of the dataset or topic, e.g., "sales performance across regions" or "student academic trends"].

## 🧰 Tools Used

- **Power BI Desktop**
- **Microsoft Excel** (for data source)
- **DAX** (Data Analysis Expressions)
- **Power Query** (for data transformation)

## 📈 Dashboard Features

- Interactive card visuals for key metrics
- Slicers for dynamic filtering by category/date
- Trend analysis using line and bar charts
- Image integration for enhanced storytelling
- Table to view all the data in tabular form.
## 🚀 How to Use

1. Download the `.pbix` file from the `dashboard/` folder.
2. Open it in **Power BI Desktop**.
3. If prompted, update the data source path to match your local Excel file location.
4. Explore the dashboard using filters and visuals.
