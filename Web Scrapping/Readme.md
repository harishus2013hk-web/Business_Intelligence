# Cricket Statistics Dashboard - JJ Bumrah vs Australia (2017–2023)

## 📘 Overview
This dashboard presents batting and bowling performance against Australia.  
The data was **collected directly from the ESPN Cricinfo website** and structured into a visual summary for analysis.

## 📊 Features
- **Batting Stats**: Matches, innings, runs, average, strike rate, boundaries, dismissals.
- **Bowling Stats**: Wickets, runs conceded, economy rate, strike rate, maidens, best bowling figures.
- **Interactive Slicer**: The player image and statistics panels update dynamically based on slicer selections, making the dashboard fully interactive.
- **Visual Layout**: Player photo, batting panel (orange), bowling panel (blue).

## 🛠️ Methodology
- Data gathered from ESPN Cricinfo (web source).
- Cleaned and structured the data.
- Designed into a dashboard using Power BI.
- Added slicer functionality to allow users to filter and view stats interactively.

## 📂 Folder Contents
- `data/` → Raw and cleaned datasets taken from ESPN.
- `dashboard/` → Visualization files or exported dashboard images.
- `README.md` → Documentation for this folder.
